# Practice Problem 4.2

Use for loops for all questions

1.

Create a function called _repeatLetter_ that takes in a character and an integer.
Return a String that is that letter repeated that many times.

Examples:

```
repeatLetter('a', 8)
returns: "aaaaaaaa"
```

```
repeatLetter('e', 0)
returns: ""
```

```
repeatLetter('!', 1)
returns: "!"
```

2.

Create a function called _countLetter_ which takes in a String and a character.
Return the amount of occurances of the character in the String.

Examples:

```
countLetter("hello", 'l')
returns: 2
```

```
countLetter("I don't know", 'e')
returns: 0
```

3. 

Create a function called _evenSum_ which takes in two integers as parameters.
returns the sum of all even integers between the two numbers inclusively.
Order of the integers does not matter

Examples:

```
evenSum(2, 6)
returns: 12
```
Explanation: 2+4+6 = 12

```
evenSum(0, 10)
returns 30
```
Explanation: 0+2+4+6+8+12 = 30

```
evenSum(9, 2)
returns 20
```
Explanation: 2+4+6+8 = 20
